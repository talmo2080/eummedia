# 📥 raw — 원본 자료 투입함

여기에 원본 자료를 **그냥 던져 넣으세요.** 하위 폴더 없이 플랫하게 둡니다.

- 넣을 수 있는 것: 기사, PDF, 메모, 대본, 책 노트, 캡처 텍스트, 회의록 등
- 파일명은 아무렇게나 둬도 되지만, 알아보기 쉬우면 좋습니다. (예: `2026-07-04-두피세미나-메모.txt`)

## 다음 한 마디면 끝

자료를 넣고 이렇게 말하세요:

> **"위키에 반영해줘"**

그러면 클론이 자료를 읽고 → `wiki/`에 정리 페이지를 만들거나 보강하고 → 관련 페이지끼리 링크하고 → 목차(index)와 이력(log)까지 갱신합니다.

특정 파일만 반영하고 싶으면 파일명을 함께 말하면 됩니다. (예: "`2026-07-04-두피세미나-메모.txt` 위키에 반영해줘")

> 이 폴더의 파일은 클론이 **읽기만** 하고 수정하지 않습니다. 원본은 그대로 보존됩니다.
