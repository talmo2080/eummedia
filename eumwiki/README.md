# 🧠 이음 위키 (eumwiki)

정세연의 개인 지식·노하우 위키. **Karpathy LLM Wiki 패턴**으로 운영합니다.
> 사용자는 원본 자료만 던져 넣고 질문한다. 정리·연결·목차 관리는 전부 클론(Claude)이 한다.

---

## 🔄 어떻게 돌아가나

```
 raw/  ──"위키에 반영해줘"──▶  클론이 정리  ──▶  wiki/
 (원본 던지기)                (읽기·요약·연결)     (정리된 페이지 + 목차)
```

1. **[raw/](raw/)** 에 자료(기사·PDF·메모·대본·책노트)를 그냥 넣는다. 하위 폴더 없이 플랫하게.
2. **"위키에 반영해줘"** 라고 말한다.
3. 클론이 → `wiki/`에 페이지 생성/보강 → 관련 페이지 교차링크 → `index.md`·`log.md` 갱신까지 한 번에 처리.

## 📂 구조

| 폴더/파일 | 역할 |
|-----------|------|
| [raw/](raw/) | 원본 자료 투입함 (사용자가 던져 넣음, 플랫) |
| [wiki/index.md](wiki/index.md) | 전체 목차 (클론이 자동 갱신) |
| [wiki/log.md](wiki/log.md) | 작업 이력 |
| [wiki/](wiki/) 하위 5개 분류 | 두피관리·미용 / 미디어·편집 / 개발·자동화 / 사업·기획 / 개인학습 |
| [CLAUDE.md](CLAUDE.md) | 클론 운영 규칙 |
| [_템플릿.md](_템플릿.md) | 페이지 작성 템플릿 |
